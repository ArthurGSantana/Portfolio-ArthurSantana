AdotapetsPets
Projeto feito com Angular versão 11, utilizei o Angular Material para desenvolver os componentes visuais. Desenvolvido para ser meu trabalho de conclusão de curso, elaborei o sistema para ajudar as pessoas na hora de adotar um novo pet. Implementei algumas opções adicionais como, auxílio pra quem perdeu seu pet ou achou algum e um painel somente para as ONGs parceiras utilizarem como monitoramento dos animais deles que pertecem ao nosso sistema.
